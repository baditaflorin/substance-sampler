# Substance Sampler

Browser-based photo-to-PBR texture creation for indie game and 3D artists, powered by WebGPU and WASM-ready static assets.

Live site: https://baditaflorin.github.io/substance-sampler/

Repository: https://github.com/baditaflorin/substance-sampler

Support development: https://www.paypal.com/paypalme/florinbadita

This repository is being bootstrapped for GitHub Pages-first delivery. The production application lives in `docs/` after `make build`.
